# 全局标题动画效果指南

## 🎨 动画类型总览

### 基础动画
- **flash** - 经典闪光扫过效果（默认）
- **fade** - 淡入淡出效果
- **scale** - 缩放动画效果
- **slide** - 滑动进入效果
- **none** - 无动画效果

### 扫光动画系列 ✨

#### 基础扫光
- **sweep_fast** - 快速扫光（2秒循环）
  - 适合：动感、节奏快的内容
  - 特点：快节奏，高频率扫光

- **sweep_slow** - 慢速扫光（5秒循环）
  - 适合：庄重、优雅的内容
  - 特点：缓慢优雅，持续时间长

#### 特效扫光
- **sweep_pulse** - 脉冲扫光（带呼吸效果）
  - 适合：有节奏感的内容
  - 特点：3阶段脉冲，带缩放效果

- **sweep_rainbow** - 彩虹扫光（颜色变化）
  - 适合：活泼、多彩的内容
  - 特点：红橙黄绿青蓝紫循环变色

- **sweep_wave** - 波浪扫光（渐变效果）
  - 适合：流畅、自然的内容
  - 特点：正弦波强度变化，带模糊

#### 高级扫光
- **sweep_laser** - 激光扫光（细线扫过）
  - 适合：科技、未来感内容
  - 特点：短暂强烈的青色激光

- **sweep_glow** - 辉光扫光（发光效果）
  - 适合：梦幻、温暖的内容
  - 特点：带模糊和缩放的辉光

- **sweep_neon** - 霓虹扫光（霓虹灯效果）
  - 适合：夜店、都市风格内容
  - 特点：紫红色霓虹，带阴影

- **sweep_electric** - 电光扫光（闪电效果）
  - 适合：刺激、动作类内容
  - 特点：3次快速闪烁电光

- **sweep_diamond** - 钻石扫光（闪亮效果）
  - 适合：奢华、高端内容
  - 特点：4阶段渐进闪亮

### 高级动画系列 🚀

#### 变形动画
- **rotate** - 旋转动画（3D旋转）
  - 适合：动感、立体的内容
  - 特点：Y轴旋转效果，4秒循环

- **bounce** - 弹跳动画（上下弹跳）
  - 适合：活泼、轻快的内容
  - 特点：上下弹跳，富有弹性

- **breath** - 呼吸动画（缩放呼吸）
  - 适合：平静、舒缓的内容
  - 特点：缓慢的缩放呼吸效果

- **shake** - 震动动画（快速抖动）
  - 适合：紧张、刺激的内容
  - 特点：快速抖动，8次震动

- **liquid** - 液态动画（流动变形）
  - 适合：流畅、柔和的内容
  - 特点：4阶段流动变形，带模糊

#### 元素动画
- **fire** - 火焰动画（燃烧效果）
  - 适合：热情、激情的内容
  - 特点：红橙黄火焰颜色闪烁

- **freeze** - 冰冻动画（结冰效果）
  - 适合：冷静、冰冷的内容
  - 特点：冰蓝色渐变效果

- **explode** - 爆炸动画（扩散效果）
  - 适合：震撼、爆发的内容
  - 特点：快速扩散放大效果

- **star** - 星光动画（闪烁星星）
  - 适合：梦幻、浪漫的内容
  - 特点：白色星光闪烁

#### 特殊动画
- **heartbeat** - 心跳动画（跳动效果）
  - 适合：情感、温馨的内容
  - 特点：双重跳动，红色心跳

- **ripple** - 波纹动画（水波纹）
  - 适合：平静、流动的内容
  - 特点：3阶段波纹扩散

- **ghost** - 幽灵动画（飘动效果）
  - 适合：神秘、虚幻的内容
  - 特点：透明度变化，飘渺感

## 🛠️ 使用方法

在 `config.mjs` 文件中修改 `titleAnimation` 参数：

```javascript
"history-person": {
  // ... 其他配置
  titleAnimation: "sweep_glow", // 选择你想要的动画效果
  // ... 其他配置
}
```

## 💡 技术特点

### 核心特性
- ✅ **真实扫过效果** - 所有扫光动画都实现了真正的"从左到右扫过"效果
- ✅ **智能延迟** - 基于字符在屏幕中的X位置计算延迟时间
- ✅ **全程循环** - 支持视频全程循环播放
- ✅ **关键词高亮** - 保持原有的关键词高亮功能（{{关键词}}）
- ✅ **自动适应** - 自动适应不同视频时长

### 动画参数
- **循环周期** - 每种动画都有独特的循环周期（1.5-5秒）
- **延迟计算** - 根据字符位置和视频宽度智能计算延迟
- **多阶段效果** - 复杂动画包含多个阶段的变化
- **颜色变换** - 支持颜色、透明度、模糊等多种效果

## 🔧 自定义建议

如需调整动画参数，可以修改 `lib/history-person-constants.mjs` 中的配置：

```javascript
// 扫光动画配置
SWEEP_CYCLE_DURATION: 3000, // 扫光循环时长（毫秒）
SWEEP_LIGHT_DURATION: 500,  // 扫光亮起时长（毫秒）
SWEEP_FADE_DURATION: 200,   // 扫光淡入淡出时长（毫秒）
```

或者直接在 `lib/title-animation.mjs` 中修改具体动画函数的参数。

## 🚀 快速开始

1. **选择动画** - 从上面的列表中选择合适的动画类型
2. **修改配置** - 在 `config.mjs` 中设置 `titleAnimation` 参数

## 📝 注意事项

- 🎯 **性能考虑** - 复杂动画（如liquid、rainbow）会增加渲染时间
- 🎨 **风格统一** - 建议全局标题和分镜字幕使用相同或相近的动画风格
- 📐 **字符间距** - 某些动画（如rotate）可能需要调整字符间距以获得最佳效果
- 🔄 **循环周期** - 较短的视频建议使用循环周期较短的动画
